Tools

Everything you actually need.

Eight operations that cover 95% of what people open Acrobat for. No wizard. No toolbar customization. Just the tool.

Merge PDFs
Combine any number of PDFs into one. Drag to reorder before merging.
Split PDF
Extract a page range or split every page into its own file.
Compress
Reduce file size without a cloud re-encoder. Adjustable quality target.
Reorder Pages
Drag pages into any order. Preview thumbnails before saving.
Annotate & Draw
Freehand drawing and annotation tools directly on the page.
Extract Pages
Pull specific pages out into a new PDF. Pick by number or range.
Images to PDF
Pack a folder of images into a PDF. Reorder before converting.
Find in Document
Full-text search with jump-to-match. Highlights across all pages.

Linux

x64, Wayland and X11 supported

  • .AppImage Universal, no install needed
  • .deb Debian / Ubuntu
  • .rpm Fedora / RHEL

All downloads are the latest stable release from github.com/zParik/NotAcrobat/releases. Builds are produced by GitHub Actions. Check the workflow file if you want to verify.
