QYRA — Free Offline PDF Toolkit & Adobe Acrobat Alternative

aka.  the free offline PDF toolkit and ultimate Adobe alternative they don't want you to have.

Subject is a free, open-source PDF operations suite for Windows and Linux stations. Performs merge, split, compress, annotate, and reorder operations entirely on the operator's machine. No telemetry pipeline, no cloud upload server, no subscription harvest. Files do not leave the perimeter. Not the file. Not the metadata. Not ever.

▸ Verify build authenticity

All release artifacts are produced exclusively by the public GitHub Actions workflow — no local builds, no manual uploads.

  1. Download the artifact and its .sha256 checksum from the release archive.
  2. On Linux, run sha256sum -c *.sha256. On Windows, run Get-FileHash <filename> -Algorithm SHA256 and compare the output against the published checksum.
  3. Build logs are publicly viewable at github.com/zParik/NotAcrobat/actions. Every release maps 1:1 to a workflow run.

If a download does not start, right-click the link and choose "Save Link As", or browse the full release archive directly.
